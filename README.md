# TypeScript Chatbot Demo

This project is a small chatbot widget that appears in the bottom-right corner of the page, like the support chats you see on many websites.

## Files

- `index.html` sets up the page and chatbot markup.
- `styles.css` handles the layout and widget styling.
- `src/chatbot.ts` contains the TypeScript source.
- `dist/chatbot.js` is browser-ready JavaScript so the demo can run immediately.

## Run it

You can open `index.html` directly in a browser.

If you want a local server:

```powershell
npm install
npm run start
```

If you want to rebuild the TypeScript output after editing `src/chatbot.ts`:

```powershell
npm install
npm run build
```
